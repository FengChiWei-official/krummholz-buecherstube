## Definition

**Finiteness** means the total steps are finite. It is a global property.

## E.g.

An algorithm that prints **all integer** isn't a **Finite** Algorithm.

A Brute-force algorithm to crack a 256-bit key has a **finite** number of steps, but it is **infeasible**.
1. generate all $2^{256}$ length passwords.
2. validate them all.
