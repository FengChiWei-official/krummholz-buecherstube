### 第二部分：核心考点精准解析（攻克复习盲区）

#### 1. 多少条腿保证厨师机器人稳定？
* **静态稳定（Static Stability）定义**：机器人的重心（COG）在地面上的投影，必须落在其支撑腿形成的**支撑多边形（Support Polygon）**内部。
* **复习要点**：
  * **静态站立稳定**：至少需要 **3条腿**。三点决定一个平面，形成一个稳定的三角形支撑区域。
  * **静态步行稳定（移动中稳定）**：至少需要 **4条腿（四足）**。当机器人抬起其中1条腿向前迈步时，留在地面上的3条腿依然能形成稳定的支撑三角形，保证移动过程中的每时每刻都是静态稳定的。
  * **六足（6条腿）交替三脚架步态（Tripod Gait）**：每次抬起3条腿，另外3条腿支撑，行走效率更高。
  * **厨师机器人场景**：厨师机器人在炒菜、颠勺时，手臂会产生剧烈的动态干扰力矩。为了在双臂工作时提供足够的稳定支撑面，通常需要至少 **3条腿（形成三脚架基座）** 站立；若需要稳定地边走边炒菜，则需要 **4条腿以上**。

#### 2. 导航避障：人工势场法算法（考纲中“人工视察法”应为“人工势场法”的输入笔误）
* **基本原理**：将机器人在环境中的运动看作是在虚拟势场中的运动。目标点产生**引力（Attractive Force）**，障碍物产生**斥力（Repulsive Force）**，合力决定机器人的移动方向。
* **算法核心公式（设计题必背）**：
  * **引力势场函数**：
    $$U_{att}(q) = \frac{1}{2} k_{att} \rho^2(q, q_{goal})$$
    其中 $k_{att}$ 是引力增益系数，$\rho(q, q_{goal})$ 是机器人与目标的距离。
  * **引力公式**（势场的负梯度）：
    $$F_{att}(q) = -\nabla U_{att}(q) = -k_{att} (q - q_{goal})$$
  * **斥力势场函数**（仅在障碍物影响半径 $\rho_0$ 内起作用）：
    $$U_{rep}(q) = \begin{cases} \frac{1}{2} k_{rep} \left( \frac{1}{\rho(q, q_{obs})} - \frac{1}{\rho_0} \right)^2 & \rho(q, q_{obs}) \le \rho_0 \\ 0 & \rho(q, q_{obs}) > \rho_0 \end{cases}$$
  * **算法缺陷**：**局部极小值（Local Minima）**。当引力和斥力大小相等、方向相反时，合力为0，机器人会卡住，无法到达目标点。

#### 3. CNN 与 ViT 参数量计算（高概率算法设计题）
* **CNN 卷积层参数量计算**：
  * 公式：$$\text{Parameters} = (K_w \times K_h \times C_{in} + 1) \times C_{out}$$
  * *其中 $K_w, K_h$ 是卷积核宽和高，$C_{in}$ 为输入通道数，$+1$ 代表偏置（bias），$C_{out}$ 为输出通道数（即卷积核个数）。*
* **ViT (Vision Transformer) 编码器中 QKV 参数计算**：
  * 输入特征维度为 $D$，自注意力机制中头数为 $h$，每个头的特征维度为 $d_k$（通常 $h \times d_k = D$）。
  * 产生 $Q, K, V$ 的三个线性投影矩阵 $W_Q, W_K, W_V$ 维度均相同，均为 $D \times D$。
  * **参数量**：$3 \times (D \times D + D)$ （包含偏置项）。
  * 自注意力输出的多头拼接投影矩阵 $W_O$ 维度为 $D \times D$（参数量为 $D^2 + D$）。

#### 4. 各种电机的控制区别（步进 vs 伺服 vs 无刷）
* **步进电机**：通过**输入脉冲**控制。一个脉冲转动一个步进角，属于**开环控制**。优点是控制简单、位置精确、成本低；缺点是在高速时容易**丢步（失步）**。
* **直流伺服电机**：自带**编码器（反馈元件）**。控制器根据编码器实时反馈调整输出，属于**闭环控制**。优点是速度快、力矩大、高精度、绝不丢步；缺点是系统复杂、成本高。
* **直流无刷电机（BLDC）**：利用电子换向器代替机械电刷，通过检测转子位置（霍尔传感器）来切换定子电流，寿命长、效率高，多用于调速控制。

### 第三部分：高分必背算法与推导模板（填空/算法设计直通车）

#### 1. PID 离散化位置式算法设计模板
考试若要求写出 PID 算法设计，请直接列出以下标准程序逻辑：

```python
class PIDController:
    def __init__(self, Kp, Ki, Kd, dt):
        self.Kp = Kp
        self.Ki = Ki
        self.Kd = Kd
        self.dt = dt
        self.integral = 0.0
        self.prev_error = 0.0

    def update(self, setpoint, pv):
        error = setpoint - pv  # 计算当前偏差
        self.integral += error * self.dt  # 积分项（消除稳态误差）
        derivative = (error - self.prev_error) / self.dt  # 微分项（抑制超调）
        
        # 位置式PID输出
        output = self.Kp * error + self.Ki * self.integral + self.Kd * derivative
        
        self.prev_error = error
        return output
```

#### 2. 局部坐标系到全局坐标系的变换（Homogeneous Transformation）
视觉坐标（Camera）转换到机器人局部坐标（Base）：
$$\begin{bmatrix} X_{base} \\ Y_{base} \\ Z_{base} \\ 1 \end{bmatrix} = \begin{bmatrix} R_{3 \times 3} & T_{3 \times 1} \\ 0_{1 \times 3} & 1 \end{bmatrix} \begin{bmatrix} X_{cam} \\ Y_{cam} \\ Z_{cam} \\ 1 \end{bmatrix}$$
*   $R$ 为旋转矩阵（代表方向变化），$T$ 为平移向量（代表位置偏移）。
*   注意：齐次坐标最后一行必须补 $1$。

#### 3. 强化学习：马尔可夫决策过程（MDP）与蒙特卡洛（MC）
*   **MDP五元组**：$(S, A, P, R, \gamma)$。
    *   $S$：状态空间，$A$：动作空间。
    *   $P(s' | s, a)$：状态转移概率（当前状态只与上一状态和动作有关，具有**无后效性/马尔可夫性**）。
    *   $R(s, a)$：回报函数，$\gamma$：折扣因子。
*   **蒙特卡洛算法（MC）**：无模型（Model-free）算法，不依赖环境状态转移概率。它通过**完整的回合（Episode）**实验来估算价值。
    *   价值更新公式：$$V(S_t) \leftarrow V(S_t) + \alpha (G_t - V(S_t))$$
    *   *其中 $G_t = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$ 是从时间 $t$ 开始的累积回报。*

### 第四部分：综合运用大题满分套路（25分开放大题）

开放大题通常会让你设计一个功能复杂的机器人（如“智能消杀机器人”或“无人递送小车”）。请直接套用以下四步法作答：

#### 步骤一：硬件系统组成与拓扑关系（画硬件系统图）
将硬件分为三层，并用双向箭头连接：
1.  **感知系统（输入）**：超声波传感器（测距避障）、光电编码器（电机测速反馈）、深度相机（目标检测输入）、IMU（姿态感知）。
2.  **控制与处理系统（核心）**：微控制器（MCU，负责PID控制和电机驱动） + 边缘计算单元（如Jetson Nano/树莓派，跑ViT/CNN模型和路径规划）。
3.  **执行系统（输出）**：直流伺服电机、驱动板、轮子。

#### 步骤二：软件体系架构图（感知 $\to$ 决策 $\to$ 执行）
在试卷上画出以下水平或垂直分层图：
```
 [ 环境感知 (Sensors) ]  ── 原始数据 ──>  [ 状态估计/定位 (SLAM / Encoder) ]
                                                       │
                                                    位置信息
                                                       ▼
 [ 轨迹控制 (PWM / Actuators) ] <── 速度指令 ── [ 路径规划与避障 (APF/小车运动学) ]
```

#### 步骤三：核心算法设计
1.  **视觉检测**：采用 CNN 模型对前方目标或人群进行特征提取与分类。
2.  **局部避障**：采用**人工势场法（APF）**。将终点设为引力源，将障碍物设为斥力源。通过计算合力：
    $$\vec{F}_{total} = \vec{F}_{att} + \sum \vec{F}_{rep}$$
    计算出下一时刻小车的前进方向 $\theta$。
3.  **运动学控制**：
    利用两轮差速小车运动学模型，将合力方向 $\theta$ 转化为左右轮的线速度：
    $$v_L = v - \frac{\omega L}{2}, \quad v_R = v + \frac{\omega L}{2}$$
4.  **电机闭环反馈**：利用**脉冲编码器**采集电机的实际转速，通过 **PID 线性算法**实时调整 PWM 占空比，消除电机转速的静差。

#### 步骤四：反馈与安全冗余设计（展示专业性）
*   **双重避障冗余**：当视觉算法（CNN）失效时，底层的**超声波传感器**作为硬中断，一旦距离小于安全阈值，MCU 立即停止电机，保证小车物理安全。
*   **积分防饱和**：在 PID 算法中对积分项进行限幅限值，防止电机在受阻时过载损坏。
