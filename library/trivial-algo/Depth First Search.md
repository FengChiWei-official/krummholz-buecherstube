## Definition
This page is an index that explains two distinct usages of DFS and links to the canonical notes.

### Two DFS modes

- **Connectivity DFS**: use when the goal is reachability or counting connected components (mark visited on entry). See [[Connectivity DFS]].
- **Backtracking / Full-path DFS**: use when enumerating full paths/solutions (mark/unmark around recursion). See [[Backtracking DFS]].
