## Euler Substitution I

$$
let \sqrt{ax^2+bx+c} = \sqrt{a}x + t\text{, then } x = \frac{t^2 - c}{b - a \sqrt{a} t}
$$

or
$$
\sqrt{ax^2 +b x +c} - \sqrt {a}x = \frac{bx+c}{\sqrt{ax^2 +b x +c} + \sqrt {a}x}
$$

so  $lim (\sqrt{ax^2 +b x +c} - \sqrt {a}x) = \frac{bx}{2\sqrt{a}x}$
>[!note] it is [[Hyperbolic Functions]] - 渐近线
